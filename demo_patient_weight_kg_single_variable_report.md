# Описательная статистика одной переменной

**Categories:** exploratory / single_variable

Строит вариационный ряд и подробные описательные характеристики одной числовой переменной.

Анализ рассчитывает общие, параметрические и непараметрические характеристики, а также готовит гистограмму, boxplot и QQ-график.

## Workbook variables

- Inputs: weight_kg
- Outputs: derived.weight_kg_variation_series

## Formulas

- `x(1) <= x(2) <= ... <= x(n)`
  Вариационный ряд - отсортированные наблюдения.
- `mean = sum(x) / n`
  Среднее арифметическое.
- `std = sqrt(sum((x - mean)^2) / (n - 1))`
  Выборочное стандартное отклонение.
- `SE(mean) = std / sqrt(n)`
  Стандартная ошибка среднего.
- `IQR = Q3 - Q1`
  Межквартильное расстояние.

## Interpretation

Сводка помогает оценить центр, разброс, форму распределения, выбросы и близость эмпирического распределения к нормальному.
