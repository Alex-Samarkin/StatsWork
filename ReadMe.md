# Пакет для статистического анализа на julia 

ключевая идея - простые (1-2 команды для любых процедур, единое пространство переменных (автоматический маппинг переменных на колонки загруженных датасетов, статистический анализ как единый модуль, который делится на темы и группы анализов), модули графики и отчетов 

1. **модуль** - манипуляция с данными и создание единого пространства - переменных (колонок датафреймов) 
2. **статистичечкий анализ** данных (делится на группы, а те на отдельные анализы) как минимум - генерация данных, разведочный анализ, тестирование гипотез, корреляционный и регрессионный анализ, методы снижения размерности, анализ временных серий, планирование эксперимента, нейронные сети Каждый анализ выдает структуру-результат, которые "знают" как вывести графику и как сформировать отчет (например md или аналогичный "простой") формат 
3. **визуализатор** (на базе plots и gadfly, но на выходе - "простые" процедуры, и единые на модуль настройки формата графиков (размеры, цветовое оформление и т.п.) 
4. **генератор отчетов** (формат md или аналогичный с поддержкой включения графики) основная идея: подключеннфый анализ дает result структуру и две стандартные процедуры: графический отчет, техтово-табличны отчет, которые и вызывает генератор отчетов желательно также реализовать концепцию workbook. 
 
Анализ начинается с создания workbook, наполняется манипуляциями и врезультатами, по окончании - workbook закрывается и дает логи и сборник результатов сначала подготовь структуру проекта для создания пакета, затем тестовый модуль, который этот пакет использует
